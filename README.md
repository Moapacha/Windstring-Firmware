# Windstring-Firmware
Most Recent updates for Ear Modular Windstring's firmware
Product page: https://www.earmodular.com/windstring

# V253 Functions

- **Random Selection**
  - Supports area-based selection.

- **Reset**
  - Locked channels cannot be reset.
  - When reset is at high voltage, random function is disabled.

- **Lock**
  - Short press → lock channel in ON position.
  - Long press (3s) → lock channel in OFF position.
  - Locked channels can still be reset.
